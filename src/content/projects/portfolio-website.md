## Overview

This is my personal portfolio website showcasing my work, interests, and memories. Built with modern web technologies, it demonstrates clean design principles and efficient static site generation.

## Features

- **Static Site Generation**: Built with Astro for maximum performance
- **Content Collections**: Type-safe Markdown content management with Zod validation
- **Responsive Design**: Works beautifully on all devices
- **Warm Aesthetic**: Cozy beige color palette for a welcoming feel
- **GitHub Pages**: Automatically deployed via GitHub Actions

## Technologies Used

- **Astro 5.x**: Modern static site generator
- **TypeScript**: Type-safe development
- **Pure CSS**: No framework overhead, full control
- **GitHub Pages**: Free hosting with CI/CD

## Implementation Highlights

### Content Management
The site uses Astro's Content Collections feature, which provides type-safe frontmatter validation using Zod schemas. This ensures content consistency and catches errors at build time.

### Design System
A comprehensive design system with CSS custom properties enables consistent spacing, typography, and colors throughout the site.

### Performance
Static generation ensures fast load times and excellent Core Web Vitals scores.

## Lessons Learned

Building this site taught me the power of static site generation and the importance of thoughtful content architecture. The modular component system makes it easy to maintain and extend.
